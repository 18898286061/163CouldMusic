# 仿网易云音乐

1.	使用了 jQuery 进行开发的移动端云音乐播放器
2.	使用了 MVC 的模块化思想的开发方式
3.	使用了EventHub 的开发思想处理事件
4.	包含了管理端，使用LeanCloud、七牛进行数据存储

**启动应用**
1. `npm i -g http-server`
2. `http-server -c-1`
3. `node server 8888`
4. `open http://127.0.0.1:8080/src/admin.html`
